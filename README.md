# Leads Tracker Chrome Extension

## 📝Description
Leads Tracker is a simple, lightweight Google Chrome extension designed to help you save and manage important URLs or text snippets directly from your browser. Whether you are conducting research, job hunting, or just want to bookmark specific pages quickly, this tool lets you save your current tab or manually input links with a single click. All data is saved securely in your browser's local storage.

##  Features
* **Save Current Tab:** Instantly save the URL of the active tab you are currently viewing.
* **Manual Input:** Type or paste specific text or links into the input field and save them.
* **Quick Access:** Click on any saved lead in the list to open it in a new tab.
* **Clear Data:** Double-click the "DELETE ALL" button to wipe your saved leads and clear your local storage.
* **Persistent Storage:** Your leads remain saved even if you close the extension or the browser, thanks to local storage integration.

##  Technologies Used
* HTML
* CSS
* JavaScript
* Google Chrome Extensions API (Manifest V3)

##  How to Install and Use (Developer Mode)
Since this extension is not currently published on the Chrome Web Store, you can easily install it locally:

1.  **Download or Clone the Repository:** Download the project files to your local machine.
2.  **Open Extension Management:** Open Google Chrome and navigate to `chrome://extensions/` in your address bar.
3.  **Enable Developer Mode:** Toggle the **"Developer mode"** switch located in the top right corner of the extensions page.
4.  **Load Unpacked:** Click on the **"Load unpacked"** button that appears in the top left.
5.  **Select Folder:** Browse your files and select the folder containing the extension files (the folder containing the `manifest.json` file).
6.  **Pin the Extension:** Click the puzzle piece icon in your Chrome toolbar, find "Leads tracker", and click the pin icon to keep it visible.

Enjoy tracking your leads!
